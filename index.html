Chainizer

TypeScript microservice Kubernetes blockchain Mongo AMQP

The project chainizer is "an alternative payment platform based on bitcoin, bitcoin cash, ethereum and litecoin". The source code of the platform is available on GitLab: gitlab.com/chainizer. Initially written in Java using Axon Framework, the whole platform has been re-written in TypeScript. The platform is composed of micro services bound together with AMQP queues. Each micro service is packaged in a docker image which is then shipped to a Kubernetes cluster. The project has been live for a while, but due to the cost of Google Kubernetes Engine, the hosting has been stopped.

Typescript Babel Webpack node Mocha Chai HTML5 WebRTC Socket.io

Home Cloud

Docker Swarm Ansible Bash Armbian Ceph Portainer Influxdata

The goal of the project is to create a home cloud based on Pine64 boards (like RaspberryPie boards). Initially, the home cloud was based on Kubernetes. However due to limitation of the resources, I switched to a Docker Swarm. The main features of the solution are: load balancing between nodes, a distributed filesystem, monitoring&alerts, reverse proxy and docker management. The project is based on some repo available on GitLab:

  • gitlab.com/tmorin/pine64-docker-swarm : A set of Ansible playbooks to setup a Docker Swarm cluster based on PINE A64-LTS boards.
  • gitlab.com/tmorin/pine64-image-armbian : Script to build an Armbian image for PINE A64-LTS boards.
  • gitlab.com/tmorin/pine64-docker-images : A set of docker images built for pine64 board.
  • gitlab.com/tmorin/pine64-k8s pine64-k8s: The initial attempt to install Kubernetes on PINE A64-LTS boards.
